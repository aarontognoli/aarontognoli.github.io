# aarontognoli.github.io

Sito personale statico per GitHub Pages di Aaron Tognoli.

## Pubblicazione su GitHub Pages

1. Crea un repository chiamato `aarontognoli.github.io`.
2. Carica questi file nella root del repository.
3. Vai in **Settings → Pages**.
4. Imposta **Deploy from a branch**, branch `main`, folder `/root`.
5. Il sito sarà disponibile su `https://aarontognoli.github.io`.

## Personalizzazione rapida

- Modifica `index.html` per aggiungere tutorial, premi, pubblicazioni o progetti.
- Modifica `assets/styles.css` per colori, spaziature e layout.
- Sostituisci il blocco `.avatar` con una foto, se desideri un ritratto.
